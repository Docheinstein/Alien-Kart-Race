Detailed Description

Base entity representing a kart.

This is responsible for handle stuff common between different types of karts.

void Kart::crash ( Kart *  k)

Handles the crash with the given kart.

If this kart is already crashing with the given kart, the crash will be ignored, otherwise a collision force of a magnitute depending on the weights, the speeds, the positions and the direction will be added to this kart. Note that will be handled only the crash for this kart, for handle the crash for the given kart this method must be called on it too.

Parameters
kthe kart this instance of kart is crashing with.
