DirectionalPoint ViewUtil::cameraDirectionalPoint ( const DirectionalPoint &  kartDirectionalPoint)
static

Returns the directional point of the camera for the given kart direction point.

The returned directional point will have the same direction but a different position depending on CAMERA_TILE_BEHIND_PLAYER_COUNT.
